## 目标

（2019.2.27）

- Create React App 
- React Router
- Redux
- React Hooks + HOC + Render Props

## 命令

```BASH
# 安装依赖
npm install 

# 热更新开发服务器：localhost:3000
npm start

# 使用Jest运行单元测试
npm test
npm test -- --coverage

# 打包生产文件
npm run build

# 构建显式的Webpack, Babel, ESLint等配置文件
npm run eject
```
